# Qucoon Dashboard(Unofficial)

A modern web application for managing portfolios and tracking investment packages.

## Features

- Create and manage portfolios
- View investment packages
- Authentication with protected routes
- Responsive UI (mobile + desktop)
- Clean and modern dashboard interface

## Tech Stack

- Next.js (App Router)
- TypeScript
- React
- Redux Toolkit
- RTK Query
- Tailwind CSS
- Formik
- Nice Modal
- React Hot Toast
